# Solidity Smart Contract Examples

Welcome to this organized and comprehensive repository of Solidity smart contract examples. All contracts are written using compiler version `0.8.20` or higher, fully documented, and structured to highlight best practices in Ethereum smart contract development.

---

## Repository Structure

- `arrays/`
- `calling_contracts/`
- `escrow/`
- `functions_and_value_types/`
- `inheritance/`
- `mappings/`
- `screenshot/`
- `reverting_transactions/`
- `sending_ether/`
- `smart_contracts/`
- `structs/`
- `voting/`

---

## 🛠️ Requirements & Setup

1. **Solidity Compiler (`solc`):** Version `^0.8.20` is recommended.
2. **Framework Support:** Can be integrated into Remix, Hardhat, or Foundry for testing and compilation.
